# USER_MANUAL

## 1. Getting Started | Bắt đầu sử dụng

### 1.1 Login | Đăng nhập
1. Mở trang đăng nhập.
2. Nhập email/username và mật khẩu.
3. Bấm **Login** để vào trang chủ.

### 1.2 Register | Đăng ký
1. Chọn **Register**.
2. Điền thông tin bắt buộc.
3. Xác nhận tạo tài khoản và đăng nhập.

### 1.3 Navigation | Điều hướng
- **Home:** Xem feed bài viết.
- **Explore:** Tìm nội dung theo chủ đề/từ khóa.
- **Messages:** Nhắn tin realtime.
- **Notifications:** Xem tương tác mới.
- **Profile:** Quản lý hồ sơ cá nhân.
- **Settings:** Điều chỉnh tùy chọn tài khoản.

## 2. Core Features | Tính năng cốt lõi

### 2.1 Create a post | Tạo bài viết
1. Vào Home.
2. Nhập nội dung, thêm media nếu cần.
3. Bấm đăng.
4. Bài viết xuất hiện trong feed.

### 2.2 Interact with posts | Tương tác bài viết
- Bấm **Like** để yêu thích.
- Bấm **Comment** để bình luận hoặc trả lời.
- Bấm **Save** để lưu vào Bookmark.

### 2.3 Follow users | Theo dõi người dùng
1. Mở trang profile người dùng.
2. Bấm **Follow** hoặc **Unfollow**.
3. Danh sách followers/following được cập nhật.

### 2.4 Send messages | Gửi tin nhắn
1. Mở mục Messages.
2. Chọn hội thoại hoặc tạo mới.
3. Nhập tin nhắn hoặc đính kèm file.
4. Tin nhắn được gửi realtime.

### 2.5 Manage profile | Quản lý hồ sơ
1. Vào Profile > Edit.
2. Cập nhật tên, bio, ảnh đại diện, ảnh bìa.
3. Lưu thay đổi.

## 3. FAQs

**Q: Tôi quên mật khẩu thì làm sao?**  
A: Hiện có luồng reset ở backend; nếu bạn chưa thấy nút tương ứng trên giao diện, hãy liên hệ hỗ trợ để được cấp lại quyền truy cập.

**Q: Vì sao tôi không gửi được file?**  
A: Có thể file vượt giới hạn dung lượng hoặc sai định dạng cho phép.

**Q: Tôi không nhận được thông báo mới?**  
A: Kiểm tra kết nối mạng, đăng xuất/đăng nhập lại, sau đó mở trang Notifications.

## 4. Troubleshooting | Xử lý sự cố

| Vấn đề | Cách xử lý |
|---|---|
| Không đăng nhập được | Kiểm tra đúng mật khẩu, thử đăng nhập lại sau 1-2 phút |
| Giao diện không cập nhật | Refresh trình duyệt, xóa cache phiên |
| Tin nhắn chậm | Kiểm tra kết nối mạng và thử gửi lại |
| Ảnh không hiển thị | Kiểm tra định dạng file và trạng thái upload |
